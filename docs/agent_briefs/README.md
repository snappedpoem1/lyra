# Agent Briefs Guide

This folder exists for parallel-lane coordination.

Use a brief when:

- a wave is split across multiple agents
- a lane has forbidden files or explicit validation rules
- a future wave is blocked and needs scoped prep work only

## How To Read

1. Read the repo-root `AGENTS.md`.
2. Read `../AGENTS.md`.
3. Open only the brief for your lane.

## Current Brief Types

- tandem execution protocol
- frontend provenance / UI-specific lane guidance
- future-wave scoped briefs as they are introduced

If there is no brief for your lane, default back to the repo root instructions and the authoritative docs set.
